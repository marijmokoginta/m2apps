# Changelog v1.2.6

## New Features
- **Interactive Database Setup**: Added support for interactive database configuration during installation and at runtime.
- **Configure Database Menu**: Added a new menu item in the interactive main menu to update database settings for installed applications.
- **Preset-Aware Defaults**: Automatically pre-fills database configuration based on `.env.example` for Laravel presets.
- **Multi-Driver Support**: Support for MySQL, PostgreSQL, SQLite, and SQL Server drivers.

## Improvements
- Added `env.ReadValues()` helper to efficiently read specific keys from `.env` files.
- Improved `internal/installer` with a flexible `DBSetupFunc` hook for better post-extraction configuration.
- Enhanced interactive UI with dedicated driver selection menus and clear configuration feedback.

## Internal Changes
- Refactored `internal/env` for better key-value management.
- Introduced `internal/dbsetup` package for centralized database management logic.
