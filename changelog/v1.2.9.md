# Changelog v1.2.9

## Fixes
- Preserved full `.env` contents during env injection by creating `.env` from `.env.example` when available (instead of starting from an empty file).

## Improvements (Laravel / Laravel-Inertia)
- Added `APP_ENV` selection (local/testing/staging/production) during install before database configuration.
- On app start/restart, auto-injects the current local IP (and `ip:port`) into `SANCTUM_STATEFUL_DOMAINS` without overriding existing values.
