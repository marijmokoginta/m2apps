# M2Apps v1.0.5

## Added

- Interactive main menu now includes full operational flow:
  - install, update, process manager, delete, channel, list, daemon, help, exit.
- Added final navigation prompt for terminal UX clarity:
  - `Back to Main Menu` shown after final/info-only screens.
- Enhanced application process status display with structured table output.
- Added URL inference for `web` / `server` process rows in process status output.

## Improved

- `m2apps app start <app_id>` now prints process PID details and detected URL when available.
- Process status table now includes:
  - colorized `STATUS` (running/started = green, stopped = red)
  - `URL` column for web/server processes.
- Interactive menu rendering now uses vertical spacing between menu items for readability.

## Runtime Preset Update

- Laravel runtime process preset now explicitly uses:
  - `php artisan serve --host=127.0.0.1 --port=8000`

## Version

- Application version bumped to `v1.0.5`.
