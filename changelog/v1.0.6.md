# M2Apps v1.0.6

## Added

- Dynamic runtime port allocation for app process start flow.
- Port persistence in process registry to keep stable runtime URL per app process.
- `{PORT}` placeholder support for runtime command templates and env injection.
- New network utility package for port checks and resolution:
  - `internal/network/port.go`

## Improved

- `m2apps app status <app_id>` now shows URL with better inference based on saved runtime port.
- `m2apps app start <app_id>` now prints per-process PID and URL details more consistently.
- Interactive menu clear-screen behavior now preserves final info output until user confirmation.
- Error paths in interactive flows now pause before returning to main menu.

## Runtime Preset Update

- Laravel web preset now uses dynamic port placeholder:
  - `php artisan serve --host=127.0.0.1 --port={PORT}`

## Validation

- `go test ./...`
- `go build ./...`

## Version

- Application version bumped to `v1.0.6`.
