# Changelog v1.3.2

## Fixes
- Updater: stop app processes earlier (before staging installer) to reduce Windows directory rename failures.
- Installer: add Windows rename retry during move steps to handle transient antivirus/file-locks (shared helper reused by updater).
- Windows lock handling: detect lock errors via syscall errno codes (locale-safe) and improve user guidance (Laragon/XAMPP, antivirus, Explorer, lingering processes).
- Windows update apply: replace fixed sleep after stop with active polling for directory unlock; ensure unlock-probe is safely restored or rolled back on failure.
- Laravel preset update: run `php artisan storage:link` and `php artisan optimize:clear` only in PostUpdate (final install path), not in staging.
- Update cleanup: only remove downloaded archive/temp artifacts after all post-update steps succeed; handle ENOSPC on metadata save by cleaning artifacts and retrying save once.
